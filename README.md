# Node.js File Uploader
Simple NodeJs app to upload files locally to the server 

## Overview
This Node.js project is designed to facilitate the easy upload of files, especially images, using the popular 'express-fileupload' package.
The inspiration for this project comes from the efficient and practical approach advocated by [Dave Gray](https://www.youtube.com/@DaveGrayTeachesCode).

## Features
  - File Upload: Easily upload files, with a focus on images, to the server.
  - Local Storage: Save the uploaded files locally on the server.
  - [express-fileupload](https://www.npmjs.com/package/express-fileupload) Integration: Leverage the 'express-fileupload' package for simplified file handling.

## Prerequisites
Before running the project, ensure you have the following installed:
  - Node.js
  - npm (Node Package Manager)

## Getting Started
1. Clone the repository:
```bash
git clone https://github.com/AElgamal5/nodeJS-file-upload.git
```
2. Navigate to the project directory:
```bash
cd nodeJS-file-upload
```
3. Install dependencies:
```bash
npm i
```
4. Run the application:
```bash
npm run dev
```
- or
```bash
node index.js
```
5. Open your web browser and go to http://localhost:3000/files to access the application.
